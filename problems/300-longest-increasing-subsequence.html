300. 最长递增子序列 · 双视角动画

同一个递归：dfs(i) = 以 nums[i] 结尾的最长递增子序列长度。内部 res 从 0 开始，遍历前面每个 j，若 nums[j] < nums[i] 就 res = max(res, dfs(j))，最后 return res + 1（加上自己）。因为子序列能在任意位置结尾，答案是 max(dfs(i) for i)，不是某个固定点。下面两个视角是同一份递归的两种展开方式：填表 = 自底向上，递归树 = 自顶向下 + 记忆化。

# dfs(i) = 以 nums[i] 结尾的最长递增子序列长度
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:

        @cache
        def dfs(i):
            res = 0
            for j in range(i):
                if nums[j] < nums[i]:        # nums[j] 能接在 nums[i] 前面
                    res = max(res, dfs(j))    # 取最长的前驱链
            return res + 1                    # +1 把 nums[i] 自己算上

        ans = -inf
        for i in range(len(nums)):            # 结尾可以是任意位置
            ans = max(ans, dfs(i))            # 所以对每个 dfs(i) 取最大
        return ans
填表视角就是把这份递归自底向上展开成 dp 数组（dp[i]=dfs(i)）；递归树视角就是自顶向下真正跑这份递归，@cache 把重复的 dfs(i) 挡成橙色 ↺ 节点。
