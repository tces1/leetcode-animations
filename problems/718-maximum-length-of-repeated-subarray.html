718. 最长重复子数组 · 二维 DP 填表动画

求两个整数数组 nums1、nums2 中公共的、连续的最长子数组长度。定义 dp[i][j] = 以 nums1[i-1] 和 nums2[j-1] 结尾的最长公共子数组长度；只有两元素相等时才能在左上角基础上 +1，否则连续性中断、归零：dp[i][j] = nums1[i-1]==nums2[j-1] ? dp[i-1][j-1]+1 : 0，答案取全表最大值。
动态规划五步法
① 含义
dp[i][j] = 以 nums1[i-1]、nums2[j-1] 为结尾的最长公共子数组长度。
② 递推
相等 → dp[i][j]=dp[i-1][j-1]+1；不等 → dp[i][j]=0。
③ 初始化
第 0 行、第 0 列全为 0（空前缀，无结尾可言）。
④ 遍历顺序
从上到下、从左到右（先算好左上格）。
⑤ 举例
[1,2,3,2,1] & [3,2,1,4,7] → 公共段 [3,2,1]，答案 3。
