152. 乘积最大子数组 · 双视角动画

因为负负得正，只盯最大值会漏掉「负最小值翻身」的情况。所以同时维护「以 nums[i] 结尾的最大乘积和最小乘积」。每一步有三个候选：nums[i] 自己、前面的最大 × nums[i]、前面的最小 × nums[i]；新最大取三者最大，新最小取三者最小。下面两个视角是同一份递推：滚动变量 = 只留上一格的 mx/mn；DP 填表 = 把每一格都记进 dpMax[] / dpMin[] 两张表。

# 滚动变量版 · 三候选（最直白，O(1) 空间）
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        ans = mx = mn = nums[0]
        for x in nums[1:]:
            cands = (x, mx * x, mn * x)   # 三个候选：自己、最大×x、最小×x
            mx = max(cands)
            mn = min(cands)               # 最小值是「负负得正」的后备
            ans = max(ans, mx)
        return ans
# DP 双数组版 · 把每一格都记下来（O(n) 空间，思路完全一样）
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        n = len(nums)
        dpMax = [0] * n
        dpMin = [0] * n
        dpMax[0] = dpMin[0] = nums[0]
        for i in range(1, n):
            x = nums[i]
            dpMax[i] = max(x, dpMax[i-1] * x, dpMin[i-1] * x)   # 以 i 结尾的最大乘积
            dpMin[i] = min(x, dpMax[i-1] * x, dpMin[i-1] * x)   # 以 i 结尾的最小乘积
        return max(dpMax)                                       # 答案是 dpMax 整张表的最大值
滚动变量版就是 DP 版的空间优化：注意到 dpMax[i] 只用到 dpMax[i-1] / dpMin[i-1]，于是不必留整张表，两个变量 mx / mn 滚动即可。
