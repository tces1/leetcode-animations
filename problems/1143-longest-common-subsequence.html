1143. 最长公共子序列 · 二维 DP 填表动画

求两个字符串 text1、text2 的最长公共子序列长度（子序列不要求连续，但相对顺序不变）。定义 dp[i][j] = text1 前 i 个字符与 text2 前 j 个字符的 LCS 长度；末字符相等则由左上 +1，不等则继承上、左中较大者：dp[i][j] = 相等 ? dp[i-1][j-1]+1 : max(dp[i-1][j], dp[i][j-1])，答案在右下角 dp[m][n]。
动态规划五步法
① 含义
dp[i][j] = text1[0..i-1] 与 text2[0..j-1] 的最长公共子序列长度。
② 递推
末字符相等 → dp[i-1][j-1]+1；不等 → max(dp[i-1][j], dp[i][j-1])。
③ 初始化
第 0 行、第 0 列全为 0（任一为空串，LCS = 0）。
④ 遍历顺序
从上到下、从左到右（依赖左上、上、左）。
⑤ 举例
"abcde" & "ace" → 公共子序列 "ace"，答案 3。
