118. 杨辉三角 · 逐行逐格递推动画

给定行数 numRows，生成杨辉三角的前若干行。规律：每行两端都是 1；中间每个数 = 它左上方与右上方两个数之和，即 t[i][j] = t[i-1][j-1] + t[i-1][j]。逐行、逐格填出整张表。时间 O(numRows²)、空间 O(numRows²)（即输出本身的大小）。

// Java · 逐行递推 · 时间 O(n²)、空间 O(n²)（输出本身）
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> tri = new ArrayList<>();
        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();
            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i) {
                    row.add(1);                       // 两端恒为 1
                } else {
                    row.add(tri.get(i - 1).get(j - 1) // 左上
                          + tri.get(i - 1).get(j));    // 右上
                }
            }
            tri.add(row);
        }
        return tri;
    }
}

        tri = []
        for i in range(numRows):
