72. 编辑距离 · 二维 DP 填表动画

给两个单词 word1、word2，每次可插入、删除或替换一个字符，求把 word1 变成 word2 的最少操作次数。定义 dp[i][j] = 把 word1 前 i 个字符变成 word2 前 j 个字符的最少操作数；末字符相等则直接继承左上 dp[i-1][j-1]，不等则 dp[i][j] = 1 + min(左上替换, 上删除, 左插入)，答案在右下角 dp[m][n]。
动态规划五步法
① 含义
dp[i][j] = 把 word1[0..i-1] 变成 word2[0..j-1] 的最少编辑操作数。
② 递推
末字符相等 → dp[i-1][j-1]；不等 → 1 + min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1])（替换 / 删除 / 插入）。
③ 初始化
第 0 列 dp[i][0]=i（删 i 次），第 0 行 dp[0][j]=j（插 j 次）。
④ 遍历顺序
从上到下、从左到右（依赖左上、上、左三个方向）。
⑤ 举例
"horse" → "ros" 需 3 步：horse→rorse→rose→ros。
