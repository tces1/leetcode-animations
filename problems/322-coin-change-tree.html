322. 零钱兑换 · 记忆化递归树动画

dfs(i) = 凑出金额 i 的最少硬币数。每个节点按「最后一枚硬币 c」向下分叉成 dfs(i − c)；dfs(0)=0 是凑齐（成功），子结果回传时取 min(子) + 1。同一个金额第二次出现时直接读缓存（↺），不再展开 —— 这就是记忆化把指数级递归压成线性的原因。

# @cache 必须放在方法内：coins 因用例而异，缓存不能跨用例共享
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        @cache
        def dfs(i):
            if i == 0:
                return 0                     # 凑齐,成功叶
            step = inf
            for c in coins:
                if i - c >= 0:               # 提前剪枝,不递归到负数
                    step = min(step, dfs(i - c) + 1)   # 最后一枚用 c
            return step                      # @cache: 同一个 i 只真正算一次
        res = dfs(amount)
        return -1 if res == inf else res
