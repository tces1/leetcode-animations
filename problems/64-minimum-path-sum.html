64. 最小路径和 · 二维 DP 填表动画

网格每格有非负代价，从左上角走到右下角（每步只能向右或向下），求路径上数字和的最小值。定义 dp[i][j] = 走到 (i,j) 的最小路径和；每格 = 本格代价 + 上方与左方中较小的那个，即 dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])。
动态规划五步法
① 含义
dp[i][j] = 从起点到 (i,j) 的最小路径和。
② 递推
dp[i][j] = grid[i][j] + min(上, 左)。
③ 初始化
dp[0][0]=grid[0][0]；第一行向右累加，第一列向下累加。
④ 遍历顺序
从上到下、从左到右（先算好上方和左方）。
⑤ 举例
[[1,3,1],[1,5,1],[4,2,1]] → 右下角 7。
示例 grid = 1,3,1;1,5,1;4,2,1 1,2,3;4,5,6 1,2,5;3,2,1 5 3,8,1,4;2,1,9,5;6,3,2,7
